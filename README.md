
# ⚡ Momentum — Project Management SaaS

A full-stack project management application inspired by Trello & Linear, built using the MERN stack with real-time collaboration.

---

## 🚀 Features

* 🔐 JWT Authentication (Login/Register)
* 🏢 Workspaces with role-based access
* 📁 Projects with status, priority & deadlines
* 📋 Kanban Board (Drag & Drop)
* 🃏 Task Management (comments, checklist, tags)
* ⚡ Real-time updates using Socket.IO
* 🔍 Search & filtering
* 📊 Dashboard with analytics
* 🎨 Modern UI with animations

---

## 🛠️ Tech Stack

* **Frontend:** React, Vite
* **Backend:** Node.js, Express
* **Database:** MongoDB Atlas
* **Real-time:** Socket.IO
* **Auth:** JWT

---

## ⚙️ Setup

```bash
git clone <repo>
cd momentum
npm run install:all
npm run dev
```

Create `.env` in backend:

```
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
CLIENT_URL=http://localhost:5173
```
